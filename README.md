# README
This repository contains security tools that I hope they are useful for Red Team, penetration testing, and Blue Team. 
All are pre-compiled and require no additional installation.

---
# Security tool list
## DiskEnum
*Main objective:* This tool was developed to automate the process of listing physical drives. It's quite similar to Disk Management except it's a CLI tool.
## Splitter
*Main Objective:* This tool is convenient for splitting a large file into smaller fragments.
